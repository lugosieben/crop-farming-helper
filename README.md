# Crop Farming Helper

![Works with Shaders (Iris)](https://cdn.modrinth.com/data/cached_images/bdf90ffb230c674200a9fba3bc1a811b615b015b.png)

This mod adds a configurable, fully client-side Overlay to the game, showing you if your crop farm is optimal.

|Color|Meaning|
|---|---|
|🟢Green|Perfect conditions! Grows in ~30 minutes|
|🟡Yellow|Good conditions, but could do better. Grows in ~40 minutes|
|🔴Red|Bad conditions. Crops grow very slowly|

(All growing speeds still depend on randomness aswell)

## How to optimize:

- Use alternating rows of crops. A crop surrounded by the same type grows very slowly!
- Hydrate your farmland
- Farmland on the edge grows slower, there's nothing you can do about it except placing empty farmland next to it

![Alternating rows of crops vs one crop comparison with Helper active](https://cdn.modrinth.com/data/JMuXxBCn/images/2daadcdb5fb31c8f707bc5ed137ca618c7e0dee9.png)

## Usage

Use the keybind (F8 by default) or `/cropfarminghelper` to toggle the Helper.

## Configuration

Use Mod Menu or `/cropfarminghelper config` to enter the config.
